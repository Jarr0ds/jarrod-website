Creating a Static Website

I have been working on a static website as part of my University studies none of these fancy Frameworks just plain old Static HTML, CSS, and JavaScript. This does give you an apperication of what even Node.js brings to the table in terms of Dynamic JS but nonetheless it has been intresting.

The Card Generating System

You would have noticed the cards on the home page this stored in a JSON object and loaded into HTML without hard coding values in HTML! The date time itself is using the Java Intl Relative Time Format Library, this allows me to generate a relative date time format in very little lines of code.
